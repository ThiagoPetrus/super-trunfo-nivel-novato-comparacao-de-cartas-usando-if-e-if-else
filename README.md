# Super Trunfo – Nível Novato

## Descrição
Projeto desenvolvido em linguagem C como parte do desafio **nível novato** do jogo Super Trunfo.  
O objetivo é comparar duas cartas utilizando estruturas de decisão `if` e `if-else`.

## Funcionalidades
- Cadastro de duas cartas
- Cálculo da densidade populacional
- Cálculo do PIB per capita
- Comparação de um atributo numérico entre as cartas
- Exibição da carta vencedora

## Tecnologias Utilizadas
- Linguagem C
- Git
- GitHub
